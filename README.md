# Portfolio Website – Internship Assignment

## Overview
This repository contains a **personal portfolio website** developed as part of an internship assignment.  
The project demonstrates frontend proficiency, structured UI composition, and adherence to mandatory evaluation criteria.

The application is optimized for **clarity, performance, and production readiness**.

---

## Tech Stack
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Version Control:** Git & GitHub

---

## Features

### Hero Section
- Name, role, and professional summary
- Clear navigation and call-to-action
- Fully responsive layout

### About Section
- Academic and professional background
- Internship-focused profile summary

### Skills Section
- Categorized technical skills
- Animated skill indicators
- Tools, frameworks, and technologies listed clearly

### Project Showcase (Critical Requirement)
- Project cards for academic / personal projects
- Each project includes:
  - Project title
  - **Clear 2-line description of the problem solved**
  - Technology stack
  - **Direct GitHub repository link (mandatory)**
- Content-focused cards without project images

### Contact Section
- Centralized contact information
- Email and social profile links
- Clean and accessible layout

---

## Performance & Quality
- Production build tested using Vite
- Responsive across devices
- Optimized Tailwind usage
- No unnecessary dependencies
